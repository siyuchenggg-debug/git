# 中亚城市设计咨询与商务洽谈英语

面向中信建筑设计研究总院第七设计院商务经营管理者的一对一商务英语课程。

课程以哈萨克斯坦阿拉木图赛兰区 CBD 曼哈顿城市设计项目为主线，延伸至 AI Park、EXPO 概念方案和奇姆肯特市场拓展，训练政府汇报、需求研判、方案推介、本地设计院协作及国际设计咨询合同谈判。

## 课程目标

完成六次课后，学员能够：

- 完成 30 秒、90 秒和 3 分钟高管型英文自我介绍；
- 介绍中信设计、第七设计院及中亚市场业务；
- 向政府、规划机构和开发企业汇报城市设计概念方案；
- 用英语澄清客户目标、成果要求和审批路径；
- 与当地设计院协商责任边界、标准适配和技术接口；
- 洽谈服务范围、费用、工期、付款、变更和知识产权；
- 完成英文会议纪要、跟进邮件和简版商务建议书。

## 建议教学方式

每次 90 分钟：10 分钟复盘、20 分钟专业输入、15 分钟句型操练、35 分钟真实角色扮演、10 分钟反馈与重说。

课程强调“能完成商务任务”，不以孤立背单词为目标。所有法规与合同资料仅用于语言和商务意识训练，实际签约应由当地法律、税务及注册设计机构复核。
