# Source Materials

This folder contains the latest approved course design.

The Excel file `中信设计中亚商务洽谈英语课程设计.xlsx` is the master course blueprint.

All lesson plans, handouts, slide outlines, listening scripts and role-play tasks should be generated based on this Excel file, especially the following sheets:

1. 课程优化总表
2. 详细课程设计
3. 听说训练闭环
4. 案例与迁移设计
5. 教材融合映射
6. 客户沟通版
7. 资料来源
